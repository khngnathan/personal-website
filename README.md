# personal-website

Personal academic website for **Nathan Kwan-Ho Ng**, Ph.D. candidate in
Computer Science at UMass Amherst.

**Live site:** <https://khngnathan.github.io/personal-website/>

Built with [Jekyll](https://jekyllrb.com/) and hosted on GitHub Pages.
The shared `<head>`, top navigation, and footer live once in
`_layouts/default.html`; each page is YAML front matter plus content.

## Local preview

```sh
bundle install
bundle exec jekyll serve
# then open http://localhost:4000/personal-website/
```
